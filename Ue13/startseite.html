Hier sehen Sie Aufgaben

Ein Programm ist immer entweder nur ein Client oder nur ein Server.

Adaptive Design benutzt Media Queries, Responsive Design benutzt stattdessen Flexbox.

Mit Adaptive Design kann eine Website nur für 3 Gerätegrößen optimiert werden.

Clients und Server übertragen untereinander ausschließlich Textdaten.

Mit einem Spaltenlayout lässt sich jede Website auf einem Smartphone oder Tablet darstellen.

Angenommen, es gäbe einen einzelnen "Google-Computer", der alle Google-Dienste bereitstellt. Wäre dieser ein Server oder ein Client? Begründen Sie Ihre Antwort.

Ich denke er wäre beides, da selbst Google von anderen Diensten abhängig ist. Daher wäre dieser Rechner manchmal auch ein Client, wenn zum Anfragen an Server der Raumfahrtorganisationen getätigt werden wo Sattelitendaten für Maps ursprünglich herkommen. Ein Server wäre dieser Google Rechner Sowieso, da er Anfragen der Google Nutzer/innen beantwortet.
